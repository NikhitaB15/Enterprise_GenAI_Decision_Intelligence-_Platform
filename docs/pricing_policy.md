# Global Pricing Policy & Package Tiers

## 1. Subscription Plans

### Standard Tier
- **Price**: $499/month (billed annually)
- **Support**: Web-only, 48-hour SLA
- **Features**: Core analytics, 5 user seats, Standard API limits (100k calls/mo)

### Professional Tier
- **Price**: $1,299/month (billed annually)
- **Support**: Priority email, 24-hour SLA
- **Features**: Advanced ML insights, 20 user seats, Extended API limits (500k calls/mo), Single Sign-On (SSO)

### Enterprise Tier
- **Price**: Custom (Starting at $5,000/month)
- **Support**: dedicated TAM, 4-hour SLA
- **Features**: Full GenAI Reasoning engine, Unlimited seats, Custom integrations, Dedicated infrastructure

## 2. Discounting Authority
- **Sales Rep**: Up to 10%
- **Sales Director**: Up to 20%
- **VP of Sales**: > 20% requires CFO approval

## 3. Grace Periods
- Overdue accounts have a 15-day grace period before feature restricted.
- Shutdown occurs at 45 days past due.
